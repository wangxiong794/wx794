那些年靠着坑蒙拐骗做测试的日子

2019-11-20

part 1

​ 测试之道其路漫漫而修远兮，吾将上下而求索。上需敷衍产品，下得折腾开发，写好报告，忽悠领导，实乃生存之道。

​ 懵懂无知少年时，离开博文峰，内荐面向工厂燃气表测试，纯技术派老领导任关卡人，慈祥又热心，聊得热火朝天，心中觉得甚好，已有留存之心。然包食住于厂，周无人烟，薪资三千，怒火中生，无处可发，积火成怨，妄浇于博，奈力不从心，焦头烂额。间隔三日，面于高新，亦也高薪，但需奔柬，亦包食住，工薪八千，高新五险，三月半假，如此极好。遥想当年种菜，非君之亲，不得核心，仅作狗托，心存不甘，偷鱼摸虾，落魄逃柬，铩羽而归，心存执念，回国深造，企图凯旋。而后面与九里，乃内控易是也，工薪四千，本不考虑。然博不放柬，几经辗转，回归九里，悠闲又难以言表。上班半年，妄欲涨文，谈终为绝，葱葱岁月，一年已到，工时变长，工资不涨，日拥过日。用例足千条，缺陷近千个，倒腾服务器，熟练自动化。
